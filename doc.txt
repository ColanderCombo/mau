An alternative UI for VERA boxes based on UI5 UI7.  

This only requires a single plugin , working on both UI7 and UI5 boxes and with fully responsive design ( working on Phones, tablet etc ). 
Key features:
- Just a classical plugin , no extra HW or servers
- fully featured for room devices scenes plugins
- scene editor
- user definable custom synotpic panel pages
- extensibility with javascript plugin files for dashboard & control panels
- responsive design / multiple devices
- local and remote access via UI7 MMS authentication servers
- D3JS animated graphs for device parent child , zWave neighbors, routes & quality
- Plugin management ( install,  updates, deletes )
- compatibility with most 3rd party plugins
- display of plugin dynamic custom icons, even in remote mode
- localized ( french, english ) , open to other languages as well
